    src/qsim/session 下模块：

    qsim.session.manifest¶

    qsim.session.manifest ¶

    SessionManifest dataclass ¶

    In-memory manifest for session revision metadata.

    load(path, session_id) classmethod ¶

    Load manifest from JSON file or create an empty one.

    save(path) ¶

    Persist manifest to JSON file.

    qsim.session.session¶

    qsim.session.session ¶

    Session ¶

    Versioned artifact session with manifest-based indexing.

    Example 
    from qsim.session.session import Session

s = Session.open("runs/session_a")
rev = s.commit("circuit", {"schema_version": "1.0", "qasm": "OPENQASM 3;"})
print(s.get(rev))

    commit(kind, payload, *, inputs=None, dependencies=None, tags=None) ¶

    Store an artifact revision and append metadata to session manifest.

    get(rev_id) ¶

    Resolve artifact path by revision ID.

    open(path) classmethod ¶

    Open or initialize a session directory.
