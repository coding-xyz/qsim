    		 None

    CirqPriorBuilder ¶

    Bases: IPriorBuilder

    Cirq-backed prior builder with graceful heuristic fallback.

    If cirq is unavailable, the builder still returns a deterministic prior and annotates fallback reason in metadata.

    build(syndrome, context=None) ¶

    Build prior model using Cirq when available, else deterministic fallback.

    StimPriorBuilder ¶

    Bases: IPriorBuilder

    Stim-backed prior builder with graceful heuristic fallback.

    If stim is unavailable, the builder still returns a deterministic prior and annotates fallback reason in metadata.

    build(syndrome, context=None) ¶
