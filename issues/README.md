# Issues Index

状态说明：
- `In Progress`：已立项，仍在执行。
- `Done`：按 issue 验收完成并已归档到 `issues/done/`。
- `Failed`：历史失败项，仅留档参考。
- `Template`：新建 issue 使用模板。

完成规则：
1. 代码、测试、`docstring`、`docs/` 都完成后，issue 才能标记完成。
2. `docs/site/` 视为构建产物，优先维护 `docs/src/`，并通过构建同步。
3. 标记完成后，issue 文件移动到 `issues/done/`。
